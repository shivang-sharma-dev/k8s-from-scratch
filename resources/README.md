## 🛠️ Tools to Know

| Tool | Purpose |
|------|---------|
| `kubectl` | CLI for K8s |
| `kind` | Local K8s cluster via Docker |
| `minikube` | Local single-node cluster |
| `k9s` | Terminal UI for K8s |
| `helm` | K8s package manager |
| `kubectx` + `kubens` | Switch clusters/namespaces fast |
| `lens` | GUI for K8s cluster management |
| `stern` | Multi-pod log tailing |
