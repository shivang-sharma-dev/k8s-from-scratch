# 06 — Scheduling

Covers how the Kubernetes scheduler places pods on nodes.

## Topics
- **Node Selectors** — simple label-based pod placement
- **Node Affinity** — advanced placement rules with expressions
- **Pod Affinity / Anti-Affinity** — co-locate or spread pods
- **Taints & Tolerations** — repel/attract pods to/from nodes
- **Priority Classes** — scheduling priority for pods
